Отличие интернет и службы всемирная паутина?

Интернет - глобальная сеть, состоящая из множества подключенных компьютеров и устройств, которые обмениваются данными.
Всемирная паутина - Это система взаимосвязанных гипертекстовых документов.

Что такое протокол?

Это набор правил и стандартов, которые определяют, как данные передаются и обрабатываются между устройствами в сети.

На основании каких протоколов работает интернет и www?

HTTP - используется для передачи данных между пользователем и сервером HTTPS - используется для передачи данных, но с шифрованием.

Как вы понимаете технологию запрос-ответ в системе wwW?

Пользователь вводит адрес в адресную строку, браузер отправляет запрос веб-серверу, далее сервер анализирует запрос и пользователь получет ответ, который ему был нужен.

Что такое сайт?

Сайт - веб страница, содержающая основные элементы сайта.

Что такое доменное имя сайта и адрес сайта?

Адрес сайта - освное имя сайта (может быть выбарна произвольно) Доменное имя - идёт после точки. Имя может указывать на географическое положение(.сom, .ru)

Из чего состоит сайт?

Шапка, тело, подвал.

Что такое html элемент и его структура?

HTML - язык гиперразметки для создания веб-страниц. тег открытия <>, тег закрытия </> и его содержимое <p>

Можно ли использовать HTML для стилизации сайта

нет.

Что такое самозакрывающейся элемент, когда он применяется

Элемент, который не имеет закрывающего тега, например когда мы хотим сделать перенос строки или вставляет картинку.

Что такое атрибут элемента для чего и как применяются атрибуты элементы

Атрибут - это дополнение информации о элементе. Атрибуты всегда записываются внутри тега. Например тег "а" самостоятельно не вставляет ссылку, но если использовать атрибут href, то ссылка сможет определится.

Что такое универсальные атрибуты, укажите несколько видов.

универсальные атрибуты - могут быть применены ко всем элементам и не зависят от типа элемента.
class - позволяет создать один или несколько классов для элемента, облегчает работу в css
Id - уникальный идентификатор элемента на странице.

Можно ли вставлять один html элемент в другой
